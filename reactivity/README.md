
### 섹션 4. 반응성
## 4.1 데이터의 불변성과 가변성
```
let a = {k:1}
let b = a
call by reference로 동작
불변성 (memory)

가변성 객체 (mutable) : {}, [], fn ...
```

```
let a = {k:1}
let b = {...a} (...를 전개연산자라고 한다. a를 b에 전개)
b 에 a의 값을 추가하고 다른 메모리에 할당된 객체가 됨
```

얕은복사, 깊은 복사가 있음.
```
let a = {k:1, o: {:2}}
let b = {...a}
전개 연산자로 복사를 했지만 a.o 라는 안쪽의 객체는 복사되지 않는다.
이것을 얕은 복사라고 한다.

안쪽의 객체까지 복사하기 위해선 깊은 복사를 해야한다.
반복문을 통해 복사를 해야하며, 복잡하기 때문에 외부 라이브러리를 사용해서 한다.
```
